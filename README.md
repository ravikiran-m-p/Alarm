**Developed by M. Ravi Kiran**

 **Live Website:**

A high-performance time management application built for accuracy and global synchronization.

## Features
- **Zero-Latency Clock:** Detects local time zone (IST) immediately on load.
- **Precision Stopwatch:** Uses delta-time calculation to prevent clock drift.
- **Lap Tracking:** Record multiple time stages without stopping the timer.
- **World Intelligence:** Searchable database for over 400 global time zones with DST support.
- **Conflict-Free Alarms:** Prevents duplicate alarm entries and auto-clears after ringing.
- **Industrial Timer:** Accurate countdown mechanism with custom sound support.

## Engineering Standards
- **Naming:** Written in strict snake_case for developer readability.
- **Optimization:** Single Audio Object logic to minimize hardware resource usage.
- **UI/UX:** Midnight Cyber aesthetic, fully responsive for all mobile and desktop devices.
